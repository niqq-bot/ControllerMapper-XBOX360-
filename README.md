# ControllerMapper-XBOX360
Desktop application for mapping Xbox 360 controller inputs to virtual touch controls using analog stick based input translation.

# Tech Stack
- Java 25
- Gradle
- Dear ImGui (imgui-java)
- LWJGL 3
- input4j
- Jackson

# Dependencies
- Controller input: input4j
- UI: imgui-java
- Rendering: LWJGL 3 (OpenGL + GLFW)
- JSON: Jackson
